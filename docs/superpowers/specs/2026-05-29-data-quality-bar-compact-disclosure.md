# 数据质量提示条紧凑折叠（2026-05-29）

## 背景

数据分析页顶部的数据质量提示条用于说明实时快照兜底、未分摊广告费和数据水位。它是辅助提示，不应在移动端或首屏顶部占用过多空间，避免压过实时大盘、订单分析等主流程内容。

## 需求

- 数据质量提示条默认只展示一行摘要。
- 详情水位默认折叠，通过下拉按钮展开。
- 摘要行保留状态、数据源、业务日和最关键提示文案。
- 展开后才展示订单、广告日终、广告实时、派生利润等水位。
- 不改变后端 `data_quality` 结构、状态判断、实时快照兜底或对账逻辑。

## 验收

- 黄色 `warning` 提示在默认状态下保持紧凑，不展示多行水位。
- 有水位数据时显示下拉按钮，按钮通过 `aria-expanded` 表示展开状态。
- 点击后可展开或收起详情；收起状态下详情区域保持 hidden。
- 共享组件继续服务实时大盘、订单利润和产品盈亏页面。

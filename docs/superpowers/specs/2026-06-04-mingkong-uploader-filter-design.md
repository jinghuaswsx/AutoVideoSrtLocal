# 明空素材上传者筛选设计

## 背景

`/xuanpin/mk` 的「视频素材库」和「昨天消耗前300」卡片已经展示
`上传者` 字段。运营需要按上传者缩小候选素材范围，尤其是在移动端筛选
昨天消耗前300时快速查看某个上传者的素材。

## 目标

1. 在明空选品筛选栏增加「上传者」筛选项。
2. 筛选覆盖「视频素材库」和「昨天消耗前300」两个素材卡片 tab。
3. 使用文本输入，按上传者名称做模糊匹配。
4. 筛选在后端分页前执行，返回的 `total`、分页按钮和实际卡片数量保持一致。
5. 当前筛选值同步到 URL，刷新页面后仍保留。

## 行为约定

- 前端参数名为 `uploader`，兼容后端读取 `uploader`。
- 空值表示不过滤上传者。
- 匹配字段为素材卡片已经展示的 `video_author`。
- 单日、时间范围、全部历史、全部实时搜索都应用该筛选。
- `昨天消耗前300` 归档也应用该筛选。
- 重置搜索时清空上传者筛选。

## 非目标

- 不新增上传者枚举接口。
- 不改变产品库和素材预选 tab 的数据源。
- 不修改快照生成、归档生成和明空同步任务。

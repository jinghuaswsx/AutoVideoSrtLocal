# 素材管理主图与产品名称列间距修正

- 日期：2026-05-20
- 范围：素材管理列表页 `/medias` 的产品管理表格
- 文档锚点：`AGENTS.md` 文档驱动代码、`web/static/CLAUDE.md` Ocean Blue 间距约束、`docs/superpowers/specs/2026-04-15-medias-list-table-and-edit-modal-design.md` 列表表格列定义。

## 背景

产品管理表格的“主图”列与“产品名称”列视觉上挤在一起。当前主图缩略图 `.oc-thumb-sm` 为 96px 正方形，但动态表格 colgroup 中主图列宽只有 88px，列宽小于内容宽度，导致主图向右贴近下一列。

## 目标

1. 主图列宽必须大于等于 96px 缩略图宽度加两个 16px 间距单位，给主图与产品名称之间留下约两个空格的视觉距离。
2. 产品名称列宽保持现有可读性，不改变链接、编辑入口、搜索、分页或接口行为。
3. 变更后更新 `medias.js` 静态资源版本，避免浏览器继续使用旧列宽缓存。

## 不做

- 不调整表格列顺序。
- 不改变缩略图尺寸。
- 不改素材管理 API 或数据库。

## 验收

1. `/medias` 产品管理表格中“主图”列和“产品名称”列之间有清晰间距。
2. 静态测试确认主图列宽不小于 128px。
3. 素材管理页面路由仍能未登录返回 302，不出现 500。
